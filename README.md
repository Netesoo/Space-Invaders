# Space Invaders – Java Implementation
A classic arcade game remake developed in Java. This project served as a practical exploration of Object-Oriented Programming (OOP) principles, 2D graphics rendering, and real-time event handling.

## 🚀 Key Features
Core Gameplay: Smooth player movement, shooting mechanics, and automated enemy movement patterns.

Collision Detection: Custom logic to handle interactions between projectiles and game entities.

Game State Management: Implementation of scoring systems and win/loss condition tracking.

OOP Architecture: Heavy use of inheritance and polymorphism to manage various game objects (Players, Aliens, Projectiles).

## 🛠️ Tech Stack
Language: Java

Graphics/UI: Java Swing & AWT (for rendering and keyboard input handling)

## 🏗️ Project Structure
The code is organized into logical classes to ensure maintainability and clear data flow:

Entity: The base class for all moving objects, handling coordinates and movement logic.

Player / Alien: Specific implementations defining unique behaviors for game units.

GameLoop: The engine responsible for updating game logic and rendering frames at a consistent rate.

## 🧠 Learning Outcomes
Building this project allowed me to focus on:

Memory Management: Efficiently handling object lifecycles within the game environment.

Real-time Input Handling: Managing concurrent keyboard events for a responsive user experience.

Code Analysis: Identifying and debugging edge cases in collision logic and rendering performance.

## 🏃 How to Run
Clone the repository.

Open the project in your preferred IDE.
Compile and run the Main.java file.
